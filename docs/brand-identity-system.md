# Alesvia — Brand Identity System

## Purpose

This document defines the first concrete visual identity system for Alesvia.

It turns the verbal and strategic brand work into usable design rules and assets:

- symbol
- wordmark
- lockup
- icon logic
- color hierarchy
- typography guidance
- usage rules for web and presentation materials

This is a first institutional identity pass, not the final lifetime brand book. It is designed to be strong enough for launch, deck use, public publishing, and early partnership materials.

- [brand-strategy.md](./brand-strategy.md)

## Identity summary

### Brand position

Alesvia should look like a modern public-interest institution, not a startup product and not a generic nonprofit.

The identity therefore needs to communicate:

- institutional seriousness
- clarity
- composure
- intellectual depth
- human-centered governance

### Core visual idea

The mark is built around an **architectural A** that also reads as a **threshold** or **portal**.

That is deliberate.

The symbol should suggest:

- an institution with structure
- a passage or path forward
- an entry into judgment rather than automation by default
- a centerline of agency inside a larger system

In brand terms:

- the outer form is the institution
- the inner gold path is autonomy

This makes the mark more meaningful than a generic monogram while staying disciplined enough for serious use.

## Logo architecture

### Primary symbol

The primary symbol is a pointed architectural A-form with a central vertical path.

Symbol meaning:

- the A anchors the organization name
- the architectural silhouette gives it institutional weight
- the central path suggests agency, passage, and the human baseline inside larger systems

### Wordmark

The wordmark is simply `Alesvia` set in a refined serif style.

Rules:

- no decorative tagline inside the logo
- no period in the formal wordmark
- use the symbol + wordmark lockup for formal brand applications
- use the symbol alone for icons, avatars, and compact applications

### Lockup

The preferred lockup is horizontal:

- symbol on the left
- wordmark on the right

This is the most usable form for:

- navbar branding
- deck covers
- one-pagers
- partner documents
- social headers

## Color system

The first-pass brand palette is:

### Primary ink

- `#16324F`
- use for wordmark, main headings, institutional surfaces, and anchor elements

Why:

- darker and more civic than a generic tech navy
- trustworthy without feeling corporate

### Accent gold

- `#C79B3B`
- use for the autonomy path inside the symbol and selected emphasis

Why:

- suggests value, signal, and centerline
- works as a serious accent rather than a decorative luxury color

### Paper

- `#F4F1EA`
- use for icon tiles, light backgrounds, and restrained warm surfaces

Why:

- warmer and more editorial than flat white
- aligns better with an institute / publishing tone

### Supporting neutrals

- text charcoal: `#221E22`
- muted neutral: `#857E8B`
- aubergine support: `#31263E`

These are already close to the existing site palette and remain usable across the interface.

## Typography

### Role split

Use typography with a clear institutional distinction:

- **Serif** for identity, major statements, and thesis-led emphasis
- **Sans-serif** for interface copy, navigation, metadata, and body structure

### Current web pairing

The site currently uses:

- Playfair Display
- Inter

That pairing is acceptable for the first brand pass because it supports:

- editorial seriousness
- readable interface structure
- a clear distinction between institution and utility

### Typographic character

The brand should feel:

- editorial
- composed
- confident

It should not feel:

- startup-minimal
- overly ornamental
- luxury-brand theatrical

## Symbol rationale

The symbol is intentionally not:

- a literal shield
- a generic abstract AI mark
- a chat bubble
- a spark / star / brain / circuit motif

Those directions would weaken the institutional and public-interest framing.

The chosen direction is better because it is:

- rooted in the letter A without becoming typographically obvious
- architectural rather than app-like
- symbolic of passage and governance rather than product features
- compact and strong at favicon scale

## Usage rules

### Preferred uses

- use the **horizontal lockup** for formal brand appearances
- use the **symbol alone** for favicon, app icon, avatar, and small UI placements
- use the **wordmark alone** only when space is wide and the symbol is already established nearby

### Background guidance

On light backgrounds:

- use primary ink for the outer form
- use accent gold for the central path

On dark backgrounds:

- place the symbol on a paper tile or use a light background container
- avoid dropping the symbol directly into busy atmospheric gradients without enough contrast

### Scaling rules

At small sizes:

- use the symbol only
- avoid tiny wordmarks

At medium and large sizes:

- prefer the full lockup

### What to avoid

- do not stretch the symbol vertically or horizontally
- do not recolor the symbol arbitrarily
- do not replace the gold centerline with gradients or rainbow accents
- do not use the mark as a decorative pattern until the identity is more mature

## Final test

The identity is working if it makes Alesvia feel like:

- a serious institution
- a human-centered standard setter
- a calm but opinionated public-interest actor
- a platform that can publish, convene, and guide
